# Tailwind CSS

Tailwind CSS is the utility-first styling system used by the frontend.

## Why ChainInvite Uses It

- Fast styling without separate component CSS files.
- Consistent spacing, color, and typography scales.
- Good responsive design support.

## Role in ChainInvite

- Admin dashboard styling.
- Invite page styling.
- Scanner page styling.
- Loading, success, and error states.

## Key Terms

- Utility class: a small class that applies one style.
- Responsive prefix: `md:`, `lg:`.
- State prefix: `hover:`, `disabled:`.
