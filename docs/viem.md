# viem

viem is the low-level TypeScript Ethereum client used by wagmi and Hardhat tooling.

## Role in ChainInvite

- Read contract data.
- Prepare and send transactions.
- Encode and decode ABI calls.
- Validate wallet addresses with helpers such as `isAddress`.

## Key Terms

- Public client: used for reads.
- Wallet client: used for signed writes.
- ABI: contract function and event description.
