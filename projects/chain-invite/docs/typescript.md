# TypeScript

TypeScript is JavaScript with static types. It catches many mistakes before the app runs.

## Why ChainInvite Uses It

- Safer frontend code.
- Better editor autocomplete.
- Stronger contract call typing through viem and wagmi.
- Correct handling of `bigint`, addresses, and ABI data.

## Role in ChainInvite

- Next.js pages and components.
- Hardhat tests and scripts.
- Contract data types in frontend code.

## Key Terms

- Type annotation: explicit value type.
- `type` / `interface`: shape of data.
- `bigint`: JavaScript representation for large Solidity integers.
- `.tsx`: TypeScript file containing React JSX.
