# wagmi

wagmi is the React hook layer used for wallet and contract interaction.

## Role in ChainInvite

- Connect wallet with `useAccount` and connectors.
- Check the active network.
- Read contracts with `useReadContract` and `useReadContracts`.
- Write transactions with `useWriteContract`.
- Track pending, success, and error states.

## Key Terms

- Hook: React function starting with `use`.
- Connector: wallet connection method.
- Provider: React wrapper that makes hooks work.
